AI Overview

The specific API URL for 2Factor's SMS OTP service using the automatic generation feature is:

[`2factor.in`](https://2factor.in/)

To use it, you need to replace the placeholders in the URL with your specific information:

*   **`:api_key`**: Your unique API key obtained from your 2Factor dashboard after signing up on the 2Factor.in website.
*   **`:phone_number`**: The recipient's mobile number, including the country code (e.g., `91XXXXXXXXXX` for an Indian number).

API Flow Overview

The typical integration process involves two main steps using separate endpoints:

1.   **Send OTP (AUTOGEN)**: You make a `GET` request to the URL provided above. The 2Factor system automatically generates a 6-digit OTP, sends it to the specified phone number, and returns a unique **`session_id`** in the response.
2.   **Verify OTP**: After the user enters the received OTP into your application, you verify it using the following endpoint:`2factor.in`
    *   **`:session_id`**: The session ID you received from the _Send OTP_ request.
    *   **`:user_otp`**: The one-time password entered by the user.

A successful verification request will return a status indicating success. For detailed documentation and other API options (like custom OTPs or voice OTPs), refer to the official [2Factor API Documentation on Postman](https://documenter.getpostman.com/view/301893/TWDamFGh).
